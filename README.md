# Quantum Random Number Generator

A beginner-friendly Quantum Computing project that generates random numbers using qubits and quantum superposition.

## Concept

A qubit can exist in superposition.  
By applying the Hadamard gate, each qubit has a chance to collapse into either 0 or 1 after measurement.

## Tech Stack

- Python
- Qiskit
- Qiskit Aer
- Streamlit
- Matplotlib

## Features

- Generate random binary numbers
- Convert binary to decimal
- Select number of qubits
- Simple Streamlit UI
- Basic visualization

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
